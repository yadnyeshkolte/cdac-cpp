

## Problem List

### 1. Greatest of Three Numbers
**Description:** Read three integers and print the largest among them.

**Example:**
```
Input: 5, 12, 8
Output: 12
```

---

### 2. Sum of First N Natural Numbers
**Description:** Use a while loop to compute the sum of the first N natural numbers.

**Example:**
```
Input: 5
Output: 15 (1 + 2 + 3 + 4 + 5)
```

---

### 3. Count Digits in a Number
**Description:** Input a number and print how many digits it has.

**Example:**
```
Input: 4523
Output: 4
```

---

### 4. Factorial using Loop
**Description:** Compute the factorial of a number using a for loop.

**Example:**
```
Input: 5
Output: 120 (5 × 4 × 3 × 2 × 1)
```

---

### 5. Armstrong Number
**Description:** Check whether a number is an Armstrong number (sum of cubes of digits equals the number itself).

**Examples:**
```
Input: 153
Output: Yes (1³ + 5³ + 3³ = 153)

Input: 9474
Output: Yes (9⁴ + 4⁴ + 7⁴ + 4⁴ = 9474)
```

---

### 6. Frequency of Each Digit
**Description:** Print how many times each digit (0–9) occurs in a number.

**Example:**
```
Input: 112233
Output: 
0: 0 times
1: 2 times
2: 2 times
3: 2 times
4-9: 0 times each
```

---

### 7. Count Even and Odd Digits
**Description:** Count the number of even and odd digits in a number.

**Example:**
```
Input: 4523212
Output: Even digits = 4, Odd digits = 3
```

---

### 8. Print Squares from 1 to N
**Description:** Print the squares of numbers from 1 to N where N is user input.

**Example:**
```
Input: 5
Output: 1 4 9 16 25
```

---

### 9. Power of a Number Using Loop
**Description:** Find the power of a number (a^b) using a loop.

**Example:**
```
Input: a = 2, b = 5
Output: 32
```

---

### 10. Numbers Divisible by 3
**Description:** Find all numbers divisible by 3 between 1 and N.

**Example:**
```
Input: 20
Output: 3 6 9 12 15 18
```

---

### 11. Print Factors of a Number
**Description:** Print all the factors of a given number.

**Example:**
```
Input: 12
Output: 1 2 3 4 6 12
```

---

### 12. GCD of Two Numbers
**Description:** Find the Greatest Common Divisor (GCD) of two numbers.

**Example:**
```
Input: 12 18
Output: 6
```
